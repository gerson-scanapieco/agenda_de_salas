== Agenda de Salas

Aplicativo para reservar horarios em uma sala na semana atual,
de 06:00 a 23:00, de segunda a sexta da semana atual.

URl: https://gerson-agenda-de-sala.herokuapp.com

Especificações:

* Ruby 2.2.1

* Rails 4.2.1

* Postgresql 9.3.6

* Twitter Bootstrap 3.3.4
